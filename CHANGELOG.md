# Changelog – KungRC Karaoke Queue

## v1.6.1
- Added About / Version section in Options
- Added Debug Info (JSON summary + Copy button)
- Kill switch for autoplay (30/60 minutes or off)
- Bias manager with:
  - Favorites list
  - Learned channels list
  - Export/Import bias JSON
- Improved popup search:
  - Shows search results with title + thumbnail + duration + channel
  - Add / VIP Insert / VIP+Play from popup
  - Channel bias + ONLY karaoke channels
- Smart search builder:
  - Adds karaoke/lyrics
  - Thai key hints (ต้นฉบับ, ผู้หญิง, ผู้ชาย)
  - Optional "No vocal" (instrumental mode)
  - Negative keywords to reduce MV / official / live / reaction clutter
- Queue:
  - VIP Insert and VIP Insert + Play now
  - Drag to reorder
  - Duration shown from metadata cache
- All data stored locally via chrome.storage.local. No remote server.
